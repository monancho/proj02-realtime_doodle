# 기능 명세서

## 화면 목록

| 화면 ID | 화면명 | 핵심 기능 |
|---|---|---|
| S-001 | 랜딩/로그인 | 서비스 소개, 로그인, 회원가입 |
| S-002 | 로비/방 목록 | 방 생성, 방 코드 입력, 내가 참여한 방 |
| S-003 | 방 대기실 | 참가자 목록, 사진 업로드, 업로드 현황, 채팅, 게임 시작 |
| S-004 | 라운드 진행 | 사진과 Canvas overlay, 드로잉 툴, 채팅, 타이머, 참가자 목록 |
| S-005 | 라운드 종료 | 완성 이미지 미리보기, 다음 라운드 대기 |
| S-006 | 결과 갤러리 | 라운드별 결과 확인, 다운로드 |

## 핵심 기능

### 인증

- 회원가입/로그인은 Firebase Authentication에서 처리한다.
- 클라이언트는 Firebase ID Token을 API 요청과 Socket 연결에 포함한다.
- 서버는 Firebase Admin SDK로 토큰을 검증하고 `firebaseUid`를 사용자 식별자로 사용한다.

### 방 생성/입장

- 방 생성 시 6자리 대문자/숫자 방 코드가 생성된다.
- 방장은 게임 시작 권한을 가진다.
- 입장 시 서버는 `roomCode` 유효성, 방 상태, 최대 인원 초과 여부를 확인한다.
- MVP에서는 진행 중인 방의 신규 입장을 차단한다.

### 사진 업로드

- 허용 형식은 `jpg`, `jpeg`, `png`, `webp`다.
- 파일 크기는 MVP 기준 5MB 이하로 제한한다.
- 원본 파일은 GridFS에 저장하고 `images` 컬렉션에는 메타데이터만 저장한다.

### 랜덤 라운드

- 서버는 `used=false`인 이미지 중 하나를 랜덤 선택한다.
- 선택된 이미지는 `used=true`로 변경한다.
- 선택 가능한 이미지가 없으면 게임 종료 상태로 전환한다.

### 실시간 드로잉

- Canvas 좌표는 `draw-stroke` 이벤트로 전송한다.
- payload는 stroke 단위 또는 throttle된 point batch를 사용한다.
- 서버는 같은 `roomCode`에 속한 사용자에게 브로드캐스트한다.
- 라운드가 종료된 상태에서는 드로잉 이벤트를 무시한다.

### 실시간 채팅

- 채팅 메시지는 `send-message` 이벤트로 서버에 전달한다.
- 서버는 trim 후 빈 문자열과 200자 초과 메시지를 차단한다.
- 최근 메시지는 선택적으로 MongoDB에 저장한다. MVP 기준 최근 50개 보관을 권장한다.

### 결과 저장과 다운로드

- 라운드 종료 시 최종 합성 이미지를 GridFS에 저장한다.
- MVP에서는 대표 클라이언트가 `dataURL` 또는 `blob`을 서버에 업로드하는 방식이 구현 난이도가 낮다.
- 결과 갤러리는 `roomCode` 기준으로 결과 목록을 보여준다.
- 다운로드는 `resultId` 기반 API를 통해 파일 스트림으로 제공한다.

## 게임 진행 규칙

1. 방장이 방을 생성한다.
2. 참가자가 방 코드로 입장한다.
3. 각 참가자가 사진을 업로드한다.
4. 방장이 게임 시작 버튼을 누른다.
5. 서버가 미사용 사진 중 하나를 랜덤 선택한다.
6. 선택된 사진 위에 제한 시간 동안 함께 그림을 그린다.
7. 라운드 중 사용자는 채팅으로 역할을 나눌 수 있다.
8. 타이머가 끝나면 서버는 라운드를 종료한다.
9. 완성 이미지를 저장하고 다음 사진으로 넘어간다.
10. 선택 가능한 사진이 없으면 결과 갤러리 화면으로 이동한다.

## 예외 처리

| 상황 | 처리 방식 | 사용자 메시지 |
|---|---|---|
| 로그인하지 않은 상태 | API/Socket 접근 차단 | 로그인이 필요합니다. |
| 존재하지 않는 방 코드 | 입장 실패 | 방을 찾을 수 없습니다. |
| 게임 중인 방 입장 | MVP에서는 차단 | 이미 진행 중인 방입니다. |
| 허용되지 않는 파일 형식 | 업로드 차단 | 이미지 파일만 업로드할 수 있습니다. |
| 파일 크기 초과 | 업로드 차단 | 파일 크기는 5MB 이하로 업로드해 주세요. |
| 채팅 공백 메시지 | 전송 무시 | 표시하지 않음 |
| 채팅 길이 초과 | 전송 차단 | 메시지는 200자 이하로 입력해 주세요. |
| 라운드 종료 후 드로잉 | 이벤트 무시 | 라운드가 종료되었습니다. |
| 이미지 부족 | 게임 시작 차단 | 최소 1장 이상의 이미지가 필요합니다. |
